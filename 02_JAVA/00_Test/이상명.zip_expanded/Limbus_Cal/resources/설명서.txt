준비단계 설정
1. 림버스_죄악_계산기.sql 실행 (SIN_TABLE 생성)
2. 1성수감자목록.sql 실행 (DECK의 기반이 되어줄 KEY 생성)
3. 림버스_덱_제작기.sql 실행 (DECK 생성)
SIN_TABLE : 게임에서 사용되는 객체를 저장하는 테이블
DECK : 'SIN_TABLE에 존재하는' 객체가 12개 저장되는 테이블

메뉴설명
1. 전체 인격 추출 (SIN_TABLE 의 모든 데이터를 공개)
2. 인격 추가 (SIN_TABLE 에 데이터 추가)
3. 인격 내용 편집 (SIN_TABLE의 정보 변경)
4. 덱 추출 (DECK의 모든 데이터를 공개)
5. 덱 편집 (DECK의 데이터 중 하나를 SIN_TABLE의 객체 중 하나로 변경)
6. 초기 편성 선언 (DECK의 데이터를 기반으로, 대략적인 결과를 예측하는 결과 출력)

개선예정 사항
- 인격 추가시 질문을 이중 연결리스트로 변경
- 인격 정보 변경시 자동으로 인격 No 변경
(현재는 인격 추가시 자동 생성된 No 변경 불가)
- 인격 전체 추출의 필터기능 및 오름차순 기능 추가
- 인격 전체 추출의 키워드 / 소속 필터 기능 추가
- 초기 편성 선언시, 7인 미만 덱 편성 기능 추가
- 초기 편성 선언시, 키워드와 죄악 기대 비율을 참조하여 인격을 추천하는 기능 추가. <- 최종목표
